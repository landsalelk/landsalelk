# Product Roadmap: 50 Improvements for LandSale.lk

This roadmap outlines 50 features and improvements to enhance the platform.

## üé® User Experience (UX) & Interface
- [ ] **Multi-Language Support:** Fully translate the site into Sinhala and Tamil.
- [ ] **Dark Mode:** Add a toggle for light/dark themes.
- [ ] **Progressive Web App (PWA):** Allow users to install the site as an app on mobile.
- [ ] **Interactive Map Search:** Allow drawing custom shapes (polygons) on the map to search specific areas.
- [ ] **Street View Integration:** Add Google Street View to property details.
- [ ] **Compare Properties:** Allow users to select 2-3 properties and compare features side-by-side.
- [ ] **Infinite Scroll:** Implement infinite scrolling for search results instead of pagination.
- [ ] **Skeleton Loaders:** Ensure all loading states use shimmer effects for a polished feel.
- [ ] **Sticky Enquiry Button:** Keep the "Contact Agent" button visible on mobile while scrolling.
- [ ] **Recent Searches:** Show users their recently searched locations/criteria on the homepage.

## üè† Features for Buyers & Renters
- [ ] **Virtual Tours:** Support for 360-degree image viewers or Matterport.
- [ ] **Commute Time Calculator:** Search properties based on travel time to work/school.
- [ ] **Neighborhood Analytics:** Show crime rates, flood risk, and development news.
- [ ] **Walk Score:** Display walkability scores for listed properties.
- [ ] **Nearby Amenities:** Map overlay showing schools, hospitals, and supermarkets.
- [ ] **Price History:** Show historical price trends for the area.
- [ ] **Mortgage Pre-Approval:** Connect directly with local banks for loan estimates.
- [ ] **Currency Converter:** View prices in USD/EUR/AUD for foreign buyers.
- [ ] **Notify Me:** Alerts for price drops on saved properties.
- [ ] **Notes on Listings:** Allow users to add private notes to properties they save.
