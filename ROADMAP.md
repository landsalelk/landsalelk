# Product Roadmap: Roadside to Digital

This document outlines the 50-step plan to revolutionize the Landsale.lk agent and owner experience.

## Phase 1: Smart Capture & Engagement (In Progress)
- [ ] **AI OCR:** Auto-fill phone/price from banner images.
- [ ] **WhatsApp Integration:** Share verification links via WhatsApp.
- [ ] **Basic Gamification:** Award points for new listings.

## Phase 2: Agent Gamification & Incentives
- [ ] **Leaderboard:** Display top agents by listing volume.
- [ ] **Hunter Badge:** Award badges for high achievers.
- [ ] **Cash Rewards:** Bonuses for converted listings.
- [ ] **Weekly Challenges:** Time-bound missions (e.g., "Find 5 lands in Colombo 7").
- [ ] **Conversion Rates:** Show success metrics to agents.
- [ ] **Level Up System:** RPG-style leveling for agents.
- [ ] **Team Mode:** Agency/Team based competitions.
- [ ] **Instant Feedback:** Notify agents why a listing was declined.

## Phase 3: AI & Technology Enhancements
- [ ] **Smart Image Enhance:** Auto-correct lighting and blur in photos.
- [ ] **Voice Input:** Dictate listing details.
- [ ] **Price Prediction:** AI-suggested listing price based on location.
- [ ] **Smart Location Pin:** GPS auto-tagging.
- [ ] **Blur Privacy Details:** Auto-blur faces/license plates.
- [ ] **Auto Description:** Generate descriptions from tags/location.
- [ ] **Duplicate Check:** Prevent double listings of the same property.
- [ ] **QR Code Generator:** Printable codes for physical banners.

## Phase 4: Owner Experience (UX)
- [ ] **Magic Login:** Passwordless SMS link login.
- [ ] **Comparable Price:** Show market context to owners.
- [ ] **Potential Buyers:** Show "50 people looking in this area".
- [ ] **Agent Profile:** Show agent stats/ratings to owner.
- [ ] **Video Message:** Agent personal intro video.
- [ ] **Language Toggle:** Trilingual support (Sinhala/Tamil/English).
- [ ] **Edit Option:** Allow owners to fix details before approving.
- [ ] **Decline Reason:** Collect feedback on rejection.
- [ ] **Negotiate Fee:** In-app fee negotiation.
- [ ] **One-Click Call:** Direct dial to agent.

## Phase 5: Monetization & Business Logic
- [ ] **Installment Plans:** BNPL for service fees.
- [ ] **Premium Upgrade:** Upsell "Top Ad" during verification.
- [ ] **Refund Guarantee:** Performance-based fee refund promises.
- [ ] **Legal Aid Upsell:** Cross-sell legal verification.
- [ ] **Dynamic Pricing:** Variable fees based on property value.
- [ ] **Subscription for Agents:** Tiered limits for high-volume agents.
- [ ] **Referral Code:** Owner referral incentives.

## Phase 6: Trust, Security & Miscellaneous
- [ ] **Agent Verification:** Show verified badges to owners.
- [ ] **Spam Protection:** Rate limit requests for the same property.
- [ ] **Data Privacy:** Call masking.
- [ ] **Time Stamp:** Metadata display.
- [ ] **Activity Log:** Read receipts for agents.
- [ ] **Rent vs Sale Detection:** Auto-classify listing type.
- [ ] **Commercial Properties:** Support for shops/offices.
- [ ] **Printable Agreement:** Auto-gen PDF contracts.
- [ ] **Neighborhood Guide:** Auto-fetch amenities data.
- [ ] **Community Report:** Flag sold properties.
- [ ] **Dark Mode:** UI theme support.
- [ ] **Before/After Preview:** Visual comparison of raw vs published ad.
- [ ] **Simple Terms:** Simplified legal copy.
- [ ] **Success Stories:** Social proof on landing pages.
- [ ] **Progress Bar:** Verification flow steps.
